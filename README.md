# Health Coaching Program Data Analysis Submission by Patrick Casimir

**Tools Used:** Python (Pandas, Matplotlib, Seaborn, Scikit-Learn), Excel.

### Files in this Submission:
1. **DA_Health_Coach_Assessment.py**: Script that processes all 5 datasets and generates all outputs.
2. **Casimir_Patrick_Health_Coaching_Model_ROI_Assignment.xlsx**:
   - *ROI Model & Scenarios*: Financial projections.
   - *Cohort Retention*: Cohort retention analysis for 30, 60, 90 days.
   - *Unified Analytical View*: The master integrated dataset.
3. **Charts (5 PNGs)**: Visual evidence of retention, risk tiers distribution and improvement probability, Engagement vs. Outcome" Correlation, and success predictors.
4. **Executive_Summary.pdf**: Strategic recommendations based on findings.

### How to Run:
Ensure `members.csv`, `biomarkers.csv`, `activity.csv`, `engagement.csv`, and `nutrition.csv` are in the same folder as the script. Run `DA_Health_Coach_Assessment.py` to regenerate the model and charts.
